# Aimero Maternal & Child Birth Hospital - React Version

This project is a React migration of the original HTML website for Aimero Maternal & Child Birth Hospital.

## Features

- Responsive design optimized for all screen sizes
- Modern React component architecture
- Comprehensive maternal, newborn, and pediatric healthcare services showcase
- Clean and professional medical website design
- Accessibility improvements

## Components Structure

The website has been broken down into reusable React components:

- **TopBar**: Displays contact information and social media links
- **Navigation**: Main navigation menu with hospital services
- **Hero**: Prominent call-to-action section with hospital introduction
- **Stats**: Key statistics showing hospital achievements
- **Services**: Overview of hospital services
- **Journey**: Pregnancy journey visualization
- **WhySpecialists**: Why choose Aimero and specialist doctor profiles
- **InfoGrid**: Testimonials, health education, appointments, and emergency contact
- **Footer**: Comprehensive footer with contact information and links

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the site in your browser.

## Migration Notes

- Original HTML/CSS was converted to React components
- Styles preserved using CSS modules approach
- Google Fonts imported via CSS
- Placeholder links added with proper accessibility handling
- Mobile-responsive design maintained
